# HoX Meeting Tools

Process sales calls and prep with AI agents.

## Run

```bash
python3 run.py
```

Requires [Claude Code](https://claude.ai/code) installed.

## Agents

| Agent | Input | Output |
|-------|-------|--------|
| `discovery-demo` | Call transcript | Deal audit, personas, follow-up email |
| `pre-meeting` | LinkedIn + context | Discovery questions |

## Add Agents

Drop a `.md` file in `agents/`. It auto-appears in the dropdown.
